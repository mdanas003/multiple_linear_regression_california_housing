# Multiple Linear Regression - California Housing Dataset

This project demonstrates the implementation of a Multiple Linear Regression model using the California Housing dataset from Scikit-Learn.

## Workflow
- Data Loading
- Exploratory Data Analysis
- Correlation Heatmap
- Train-Test Split
- Feature Scaling using StandardScaler
- Model Training using LinearRegression
- Evaluation using MSE, MAE, RMSE, R2
- Residual Analysis
- Model Serialization using Pickle

## Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
